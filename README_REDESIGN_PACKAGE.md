## 🎨 DESIGN SYSTEM SNAPSHOT

### Colors
```
Primary: #F97316 (Orange) - CTAs, brand accent
Success: #22C55E (Green)  - On-track, approved
Warning: #F59E0B (Amber)  - At-risk, needs attention
Error:   #EF4444 (Red)    - Behind, failed
Info:    #3B82F6 (Blue)   - Help, informational
Neutral: #1E293B–#F8FAFC  - Text, backgrounds
```

### Typography (8 levels)
```
Display      48px 700w   (Hero)
Display-2    36px 700w   (Page title)
H1           28px 600w   (Section header)
H2           24px 600w   (Card title)
H3           20px 600w   (Subsection)
H4           18px 500w   (List header)
Body         16px 400w   (Main content) ← Most used
Caption      12px 500w   (Metadata)
```

### Spacing (4px grid)
```
2px, 4px, 8px, 12px, 16px, 20px, 24px, 32px, 40px, 48px, 64px
Used for: padding, gaps, margins, and alignment
Example: 16px button padding = 8px (v) × 20px (h)
```

## 📱 RESPONSIVE STRATEGY

```
Mobile (320px)        Tablet (768px)        Desktop (1024px)
┌─────────┐           ┌────────┬────────┐   ┌─────┬─────┬─────┬─────┐
│Content  │    →      │Content │Content │   │Card│Card │Card │Card │
│(1 col)  │           │(2 col) │(2 col) │   │Card│Card │Card │Card │
│Bottom   │           │        │        │   │────────────────────────│
│Nav ⏬    │           │Tabs    │        │   │     Sidebar: ☰→[■]    │
└─────────┘           └────────┴────────┘   └─────┴─────┴─────┴─────┘

Strategy: Mobile-first → Enhance for tablet → Optimize for desktop
Touch targets: 44px minimum on mobile
Responsive patterns: Grid layouts that stack vertically → horizontally
```
